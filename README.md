Este projeto foi desenvolvido como parte da disciplina de **Design de Código** do curso de Engenharia de Software. O objetivo foi aplicar conceitos fundamentais de desenvolvimento web em uma ferramenta funcional e bem estruturada.

---

### ⚙️ Funcionalidades

O sistema oferece uma interface minimalista para o cálculo aritmético rápido:

* **🔢 Processamento Simples:** Cálculo automático da média entre dois números inseridos.
* **⚡ Resposta Instantânea:** Resultados exibidos sem a necessidade de recarregar a página.
* **📱 Responsividade:** Design adaptável, garantindo usabilidade em dispositivos móveis, tablets e desktops.

Você pode testar o sistema através do link abaixo:

---

### 🛠️ Tecnologias Utilizadas

A aplicação foi construída utilizando o trio fundamental do desenvolvimento Front-End:

* **HTML5:** Estruturação semântica e acessível.
* **CSS3:** Estilização moderna e layout responsivo.
* **JavaScript:** Lógica de cálculo e manipulação dinâmica de elementos (DOM).

---

### 🧠 Foco em Design de Código

Apesar da simplicidade da tarefa, o projeto prioriza:
1. **Clareza de Variáveis:** Código legível e fácil de manter.
2. **Separação de Responsabilidades:** Organização lógica entre estrutura, estilo e comportamento.
3. **Experiência do Usuário (UX):** Interface limpa e direta ao ponto.
